# Algorithmique et Programmation C – Exam 2025/2026 (Semester 1)

This folder contains C exercises for the IA course exam.

## Exercises

- `b_heron_suite.c` – Calculates the recursive Heron sequence up to n terms
- `c_uppercase.c` – Counts uppercase letters in a string (max 100 characters)
- `d_tableau_1d.c` – Computes average, standard deviation, and counts values above average
- `e_tableau_2d.c` – Counts repeated integer parts in a 2D array

## Compilation & Run

Example for `b_heron_suite.c`:

```bash
gcc b_heron_suite.c -o heron
./heron
