# Programmation Python – Exam 2025/2026 (Semester 1)

This repository contains Python exercises for the IA course exam (Semester 1).

## Exercises

- `exercice_b_sommation.py` – Computes the sum:
  S = 1^1 + 2^2 + ... + N^N

- `exercice_c_permutation.py` – Performs a circular permutation of three integers (x, y, z)

- `exercice_d_suite.py` – Computes and displays the terms of a sequence defined by:
  U0 = 1  
  Un = 5 * U(n-1) + 2

- `exercice_e_tableau_1d.py` – Reads a list of grades, sorts them in descending order, and displays mentions
  (Très bien, Bien, Assez bien, Passable, Ajourné)

- `exercice_f_tableau_2d.py` – Reads two 3x3 matrices and displays their sum

- `exercice_g_epicier.py` – Computes the sum and average of product prices entered by the user

## Run

Example for `exercice_b_sommation.py`:

```bash
python exercice_b_sommation.py
